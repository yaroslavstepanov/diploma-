# Field Generator Frontend

Веб-интерфейс для настройки полей таблицы и генерации синтетических данных.

## Возможности

- **Многополевой режим** — таблица с несколькими колонками (имя, тип, режим, генератор)
- Три режима генерации: Случайное, Последовательное, Фиксированный список
- Параметры генератора зависят от выбранного типа (min/max, use_float, precision и др.)
- Поддержка ClickHouse и PostgreSQL
- Предпросмотр сгенерированных данных
- Сохранение профиля как шаблона (JSON)

## Быстрый старт

```bash
# Из корня проекта
uvicorn backend.server:app --reload --port 5000
```

Откройте: **http://localhost:5000**

## Использование

1. **Добавьте поля** — кнопка «+ Добавить поле», заполните имя, тип, режим, генератор
2. **Настройте параметры** — выберите строку, в панели справа задайте параметры генератора
3. **Подключение к БД** — укажите хост, порт (18123 для ClickHouse, 5433 для PostgreSQL), учётные данные
4. **Сгенерировать** — укажите таблицу и количество строк, нажмите «Сгенерировать в БД и показать предпросмотр»

## API

Фронтенд загружает справочники при старте:
- `GET /api/generators` — список генераторов и их параметров
- `GET /api/supported-types` — поддерживаемые типы (String, Int32, DateTime, UUID)

Генерация: `POST /api/generate` с телом `{ fields: [...], connection, target_table, rows, ... }`
